# crustacean-capital

A transactional API exclusively for crab people (those dabbling in Rust).

## Description

A simple web server written in Rust.

## Getting started

Simply run it with:
```
cargo run
```
The implemented endpoints are:

| Action | Path | Description |
|---|---|---|
| GET | /users | get all users |
| POST | /users | |
| GET | /accounts | |
| POST | /accounts | |
| GET | /transactions | |
| POST | /transactions | |
